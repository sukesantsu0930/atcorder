C - Colorful Beans 解説 /

実行時間制限: 2 sec / メモリ制限: 1024 MB

配点 : 250250250 点

問題文

NNN 種類のビーンズが 111 粒ずつあります。 iii 種類目のビーンズはおいしさが AiA_iAi​ で色が CiC_iCi​ です。ビーンズは混ぜられており、色でしか区別することができません。

あなたはビーンズの色を 111 つ選び、その色のビーンズをどれか 111 粒食べます。ビーンズの色をうまく選ぶことで、食べる可能性のあるビーンズのおいしさの最小値を最大化してください。

制約

  • 1≤N≤2×1051 \leq N \leq 2 \times 10^{5}1≤N≤2×105
  • 1≤Ai≤1091 \leq A_i \leq 10^{9}1≤Ai​≤109
  • 1≤Ci≤1091 \leq C_i \leq 10^{9}1≤Ci​≤109
  • 入力は全て整数である。

入力

入力は以下の形式で標準入力から与えられる。

NNN
A1A_1A1​ C1C_1C1​
A2A_2A2​ C2C_2C2​
⋮\vdots⋮
ANA_NAN​ CNC_NCN​

出力

食べる可能性のあるビーンズのおいしさの最小値の最大値を整数として出力せよ。

入力例 1Copy

Copy
4
100 1
20 5
30 5
40 1

出力例 1Copy

Copy
40

同じ色のビーンズは互いに区別がつかないことに注意してください。

選ぶことができる色は 色 111 と 色 555 です。

  • 色 111 のビーンズは 222 種類あり、美味しさはそれぞれ 100,40100, 40100,40 です。よって、色 111 を選んだときのおいしさの最小値は 404040 です。
  • 色 555 のビーンズは 222 種類あり、美味しさはそれぞれ 20,3020, 3020,30 です。よって、色 555 を選んだときのおいしさの最小値は 202020 です。

おいしさの最小値を最大化するには 色 111 を選べばよいため、そのときの最小値である 404040 を出力します。

92 4

出力例 2Copy

Score: 250250250 points

Problem Statement

There are NNN types of beans, one bean of each type. The iii-th type of bean has a deliciousness of AiA_iAi​ and a color of CiC_iCi​. The beans are mixed and can only be distinguished by color.

You will choose one color of beans and eat one bean of that color. By selecting the optimal color, maximize the minimum possible deliciousness of the bean you eat.

Constraints

  • 1≤N≤2×1051 \leq N \leq 2 \times 10^{5}1≤N≤2×105
  • 1≤Ai≤1091 \leq A_i \leq 10^{9}1≤Ai​≤109
  • 1≤Ci≤1091 \leq C_i \leq 10^{9}1≤Ci​≤109
  • All input values are integers.

Input

The input is given from Standard Input in the following format:

NNN
A1A_1A1​ C1C_1C1​
A2A_2A2​ C2C_2C2​
⋮\vdots⋮
ANA_NAN​ CNC_NCN​

Output

Print as an integer the maximum value of the minimum possible deliciousness of the bean you eat.

Sample Input 1Copy

Copy
4
100 1
20 5
30 5
40 1

Sample Output 1Copy

Copy
40

Note that beans of the same color cannot be distinguished from each other.

You can choose color 111 or color 555.

  • There are two types of beans of color 111, with deliciousness of 100100100 and 404040. Thus, the minimum deliciousness when choosing color 111 is 404040.
  • There are two types of beans of color 555, with deliciousness of 202020 and 303030. Thus, the minimum deliciousness when choosing color 555 is 202020.

To maximize the minimum deliciousness, you should choose color 111, so print the minimum deliciousness in that case: 404040.
